1. Установите необходимые зависимости
Необходимо установить сам python если не скачан https://www.python.org/downloads/  ВАЖНО когда запускаете файл установки питона там будет пункт ADD TO PATH нужно что-бы там была галочка.
Скрипт использует библиотеку pycaw для управления громкостью.зайдите в cmd и установите её командой:
pip install pycaw

Откройте терминал (CMD, PowerShell или Bash) и выполните:
(Если скрипт находится на рабочем столе, иначе путь меняется)
cd Desktop
python script.py
